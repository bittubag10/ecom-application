# E-Commerce Application (IMCA Project)



## ✨ Features
* **User Management:** User registration aur details.
* **Product Catalog:** Products ko manage karna aur stock update karna.
* **Cart System:** Items add karna aur quantity manage karna.
* **Order Management:** Order place karna aur Order History dekhna.
* **Monitoring:** Spring Boot Actuator se health check aur metrics.

## 🛠️ Tech Stack
* **Language:** Java 21
* **Framework:** Spring Boot 3.x
* **Database:** H2 (In-memory) / MySQL
* **Tools:** Lombok, Maven, Git

## 🚀 Kaise Run Karein?
1. Repository ko clone karein.
2. Maven dependencies install karein: `mvn clean install`
3. Application run karein.
4. Postman se APIs test karein.
